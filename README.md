# shuttle后台管理系统

# 依赖

* JavaScript框架：Vue.js
* UI: Element UI
* 路由：Vue Router
* 全局数据管理：Vuex
* 请求客户端：axios

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
